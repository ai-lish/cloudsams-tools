slp-split-pdf

stable

Split CloudSAMS SLP PDF into per-page / per-range files

cd tools/slp-split-pdf
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
../../cloudsams slp-split-pdf --input slp.pdf --output-dir ./output

How tools fit together in 10 seconds

Dispatcher（cloudsams_tools/）零第三方依賴，靠 runtime 掃 tools/*/tool.yaml 嚟做到 discoverability，冇中央 registry。

  1. 每個 tool folder 有一個 tool.yaml（name / summary / status / runtime）— cloudsams list 淨係讀呢個檔，唔 import tool 本身。
  2. 每個 tool 有一個 main.py，入面實作 run(argv) -> int —— 呢個係唯一嘅執行 contract。
  3. cloudsams <name> ... 先 lazy import 嗰一個 tool；壞咗一個 tool 唔會拖冧其他 tool 或 list。

Quickstart

用統一 CLI 定係獨立行個 tool 都得，睇你順手。

  1. Clone repo，用統一 CLI 睇齊所有工具（唔使裝任何 dependency）： 
    git clone https://github.com/ai-lish/cloudsams-tools.git
cd cloudsams-tools
./cloudsams list
  2. 裝某個 tool 自己嘅依賴，再經 dispatcher 行： 
    cd tools/slp-split-pdf
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
../../cloudsams slp-split-pdf --help
  3. 或者完全獨立行，唔經 dispatcher： 
    python split.py --input slp.pdf --output-dir ./output
